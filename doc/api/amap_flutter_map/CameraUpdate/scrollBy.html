scrollBy method

CameraUpdate scrollBy (
  1. double dx,
  2. double dy
)

按照指定到像素点dx和dy移动地图中心点

dx是水平移动的像素数。正值代表可视区域向右移动，负值代表可视区域向左移动

dy是垂直移动的像素数。正值代表可视区域向下移动，负值代表可视区域向上移动

返回包含x，y方向上移动像素数的cameraUpdate对象。

Implementation

  return CameraUpdate._(
    <dynamic>['scrollBy', dx, dy],
  );
}
